## Architecture

This is a Preact tutorial app that demonstrates Preact features through interactive examples. Each feature gets its own page, navigable via a sidebar.

**Stack:**
- [Preact](https://preactjs.com/) — React-compatible UI library (`preact/hooks`, `preact/compat` via jsxImportSource)
- [`@preact/signals`](https://preactjs.com/guide/v10/signals/) — reactive signal primitives (`signal`, `computed`, `effect`, `batch`)
- [TanStack Router v1](https://tanstack.com/router) — file-based routing with `@tanstack/router-plugin/vite` for auto code-splitting

**Routing:**
- Routes live in `src/routes/`. Each file exports a `Route` constant created with `createFileRoute('/path')`.
- `src/routeTree.gen.ts` is auto-generated by the Vite plugin on `dev`/`build` — never edit it manually.
- The root layout (`src/routes/__root.tsx`) renders the sidebar nav + `<Outlet />`.
- `src/main.tsx` bootstraps via Preact's `render()` (not ReactDOM), wrapping `<RouterProvider>`.

**Adding a new tutorial page:**
1. Create `src/routes/my-topic.tsx` with `export const Route = createFileRoute('/my-topic')({ component: ... })`.
2. The router plugin regenerates `routeTree.gen.ts` automatically on next dev/build run.
3. Add a `{ to: '/my-topic', label: 'My Topic' }` entry to the `navLinks` array in `src/routes/__root.tsx`.

**JSX config:** `tsconfig.json` sets `"jsxImportSource": "preact"`, so JSX compiles to Preact's h/Fragment without explicit imports. Hooks come from `preact/hooks`; context from `preact`.
